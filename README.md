# 📝 To-Do List Website

A modern and user-friendly To-Do List web application that helps users organize, manage, and track their daily tasks efficiently. This project is built using HTML, CSS, and JavaScript and focuses on providing a simple yet productive task management experience.

---

## 🚀 Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- 📊 Task completion progress (0–100%)
- 📝 Add notes to each task
- 🎨 Multiple themes (Light, Dark, and Custom)
- 😊 Emojis and icons for tasks
- 🌈 Color-coded priority levels
  - 🔴 High
  - 🟠 Medium
  - 🟢 Low
- 📅 Calendar view
- 📂 Create sections and subsections
- 📌 Pin important tasks
- ⭐ Favorite tasks
- 🔍 Search tasks
- 🎯 Filter tasks by status and priority
- 📆 Due dates
- 🔁 Recurring tasks
- 📈 Dashboard showing task statistics
- 📱 Responsive design for desktop, tablet, and mobile

```

3. Open `index.html` in your web browser.
